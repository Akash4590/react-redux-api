# React Practice Repository

This repository contains my **daily practice projects in React.js**, focusing on REST APIs, CRUD operations, form validation, lazy loading, and modern React patterns. The goal is to strengthen my frontend development skills and prepare for real-world React projects.

---

## 🚀 Topics Covered

### 1. REST API Integration
- Fetch data from external APIs using `fetch`.
- Display API data in React components.
- Handle asynchronous operations with proper loading states.

### 2. CRUD Operations
Implemented full REST API operations using **DummyJSON Users API**:
- **GET** – Fetch and display user data.  
- **POST** – Add new users through forms.  
- **PUT** – Update existing user data.  
- **DELETE** – Remove users from the API.  

### 3. Form Validation
- Controlled components with React state.
- Input validation for forms.
- Display error messages for invalid inputs.
- Practice of `useState` and `useEffect` for dynamic form handling.

### 4. Lazy Loading
- Implemented **on-demand component loading** using `React.lazy()` and `Suspense`.
- Optimized performance by loading heavy components only when needed.

### 5. Suspense & Experimental `use()` Hook
- Explored React **experimental features**:
  - `use()` hook for handling Promises in components.
  - Suspense for asynchronous data fetching.
